# Go-Hounds
Go Hounds web application for Loyola University Maryland's Athletic Department!

# Web Programming Assignment #1 
We implemented the main page, including an introduction and links to subsequent pages including game schedules, prizes students can earn from earning points, as well as an example of student points. We used CSS and Bootstrap to create an aesthetic and responsive design!

# Web Programming Assignment #2
We implemented a way for students to check into the games, and a way for them to view and hide prizes! We also added a pop up window which would allow students to be re-directed to a different page to either submit another form, or be directed to homepage!
